# Prompts pour les 24 Cartes de Bascule
## Cadavre Exquis des Futurs — Horizon 2036

---

## Instructions générales (à inclure dans chaque prompt)

```
Style: Retro-futuristic isometric illustration, clean lines, diagrammatic style, vintage infographic feel.
Format: Portrait A5 card layout with space for text overlay.
The card must include:
- A title banner at the top
- The main descriptive text in large readable font
- A small category label in one corner
- The illustration as the central visual element
Background: Aged, textured cream/off-white paper.
Typography: Sans-serif geometric font (like Montserrat) for title, classic serif for body text.
```
